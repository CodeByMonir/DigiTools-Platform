## 📌 Project Name

**Digital Tools Platform**

---

## 🧾 Project Description

This is a responsive digital tools marketplace built with React where users can explore different digital products and add them to a cart. The project focuses on clean UI, smooth interaction, and practical e-commerce functionality like cart management, product toggling, and checkout behavior.

Users can switch between product view and cart view, add/remove items, and see real-time updates in the cart with toast notifications for better user feedback.

---

## 🛠️ Technologies Used

* React.js
* Tailwind CSS
* DaisyUI
* JavaScript (ES6+)
* React-Toastify
* JSON (for product data)

---

## ✨ Key Features

### 1. 🛒 Dynamic Cart System

* Add products to cart
* Remove individual items
* Auto update total count in navbar
* “Proceed to Checkout” clears the cart

### 2. 🔄 Product & Cart Toggle

* Toggle between **Products** and **Cart** view
* Default view shows products
* Cart shows empty message initially

### 3. 🔔 Toast Notifications

* Shows alerts when:

  * Product added
  * Product removed
  * Checkout completed

---

## 🎯 Additional Features

* Button state change (Buy Now → Added)
* Fully responsive design (mobile, tablet, desktop)
* Clean UI based on Figma design
* Organized reusable components
